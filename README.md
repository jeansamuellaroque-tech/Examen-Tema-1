# 🚢 Proyecto API Canvas 2D – Dibujo de Barco

## 📌 Descripción

Este proyecto consiste en el desarrollo de una aplicación web que utiliza la **API Canvas 2D de HTML5** para dibujar por programación la figura de un barco, utilizando al menos **30 figuras básicas**.

La aplicación muestra:

- 🖼️ A la izquierda: El dibujo generado mediante programación.
- 🖼️ A la derecha: La imagen original de referencia.
- 🎨 Diseño moderno utilizando Bootstrap 5.
- 📄 Código organizado por tipo de figura.
- ⭐ Favicon personalizado relacionado con el proyecto.

---

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- API Canvas 2D
- Bootstrap 5

Aplicar los conceptos de graficación utilizando figuras básicas como:

- Cielo

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en un navegador web.
3. El dibujo se genera automáticamente al cargar la página.
